# Women in DeFi Enugu — Onboarding Portal

The official onboarding portal for the Women in DeFi Enugu chapter. A structured, beginner-friendly Web3 learning platform with interactive modules, quizzes, cohort registration, and a curated resource library — built to lower the barrier to entry for women in Eastern Nigeria.

**🔗 Live Demo:** https://wid-onboarding-portal.vercel.app/

---

## Features

### Learn
- **4 interactive modules** covering the complete DeFi entry journey:
  - Module 1: Set Up Your Wallet — what a wallet is, how to create one, seed phrase safety
  - Module 2: Understanding DeFi — DEXs, lending protocols, yield strategies, risk basics
  - Module 3: Essential Tools — DefiLlama, Etherscan, CoinGecko, Bubblemaps, and more
  - Module 4: Staying Safe — common scams, smart contract safety, how to start small
- **Progress tracker** — visual progress bar showing modules completed
- **Quizzes** — multiple choice questions at the end of each module to test understanding
- **Curated resources** — links to official tools and guides for each topic
- **Module navigation** — jump to any module at any time; completed modules marked with a tick

### Register
- **Cohort registration form** — name, email, Twitter handle, and DeFi experience level
- **Experience selector** — from Complete Beginner to Fairly Experienced
- **Confirmation screen** — friendly success state on submission

### Resources
- **Resource library** — 6 curated materials including guides, checklists, tutorials, video recordings, and community links
- Clean card layout with category tags and external link indicators
